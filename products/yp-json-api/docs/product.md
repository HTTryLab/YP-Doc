# YP.JsonApi

一个功能强大的 .NET 动态 JSON API 框架，允许您通过配置文件快速定义和管理 API 端点，无需编写繁琐的控制器代码。

## 核心特性

- **动态 API 注册**：通过 JSON 配置文件定义 API 端点
- **自定义处理器**：支持灵活的动态 API 处理器实现
- **Swagger 集成**：自动生成 API 文档
- **JWT 认证**：内置认证和授权支持
- **全局变量**：提供全局变量服务
- **CORS 支持**：灵活的跨域配置
- **可扩展性**：通过扩展方法轻松集成到 ASP.NET Core 项目中

## 技术栈

- **.NET 10.0**：最新的 .NET 框架
- **ASP.NET Core**：现代化的 Web 开发框架
- **Swashbuckle**：Swagger UI 集成
- **Microsoft.IdentityModel**：JWT 认证支持

## 快速开始

通过简单的几行代码，即可在您的 ASP.NET Core 项目中集成 YP.JsonApi。
